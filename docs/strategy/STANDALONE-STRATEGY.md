# Standalone Strategy

This is the strategy entry doc for Rune's standalone-first product and deployment posture.

## Core position

Rune is designed as a standalone-first personal AI runtime.

That means:
- a single-machine operator path is first-class
- local durability and inspectability matter
- broader server-grade deployment remains supported
- distributed complexity should justify itself instead of being assumed by default
